# Fundamentos de Segurança da Informação

 Objetivo
Este repositório tem como objetivo documentar estudos e aprendizados sobre os fundamentos da Segurança da Informação, servindo como base para formação técnica e profissional.

 Conteúdos Abordados
- Conceitos básicos de Segurança da Informação
- Tríade CIA (Confidencialidade, Integridade e Disponibilidade)
- Gestão de riscos
- Ameaças, vulnerabilidades e impactos
- Controles de segurança

 Metodologia
- Estudos teóricos
- Anotações técnicas
- Exemplos práticos
- Análises de cenários

 Atividades
- Estudos de casos
- Simulações teóricas
- Documentação de aprendizados

Aviso Ético
Este repositório é destinado exclusivamente para fins educacionais.
Nenhum conteúdo aqui deve ser utilizado de forma indevida ou sem autorização.
