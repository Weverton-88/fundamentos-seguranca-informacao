Controles de Segurança da Informação 

Controles de segurança são medidas adotadas para proteger os ativos de informação,
reduzindo riscos e garantindo a confidencialidade, integridade e disponibilidade.

 Tipos de Controles

 Controles Preventivos
Visam impedir a ocorrência de incidentes de segurança.
- Controle de acesso
- Autenticação
- Políticas de segurança
- Firewall

 Controles Detectivos
Permitem identificar incidentes quando eles ocorrem.
- Monitoramento
- Logs
- Sistemas de detecção de intrusão (IDS)

 Controles Corretivos
Atuam após a ocorrência de incidentes para minimizar impactos.
- Backup
- Recuperação de sistemas
- Planos de resposta a incidentes

 Importância
A aplicação adequada de controles de segurança reduz vulnerabilidades e fortalece a postura de segurança da organização.
